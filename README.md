# Utilities

## JS
  [Color.js](https://github.com/noenarcisse/utils/blob/main/js/Color.js) - Color class<br />
  [perl.utils.js](https://github.com/noenarcisse/utils/blob/main/js/perl.utils.js) - Perl utilities, recreate and extends some Perl principles such as String incrementation (or decrementation)<br />

## Perl
  [renamer.pl](https://github.com/noenarcisse/utils/blob/main/pl/renamer.pl) - Mass renamer for the files in the current directory, in Perl<br />
```bash
perl renamer.pl searchArg [replaceArg]
```
